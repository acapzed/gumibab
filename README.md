브라우저 아이콘 한 번 클릭으로 **PlanEAT 구미 2C 3식당**의 당일 메뉴를 바로 확인하는 Chrome 확장 프로그램입니다.

![스크린샷](src/screenshot_webstore.jpg)

## 주요 기능

- 중식 / 석식 탭 전환
- 이전 / 다음 날짜 탐색
- 메뉴 이미지 클릭 시 확대 보기
- 칼로리, 탄수화물, 단백질, 지방 영양 정보 표시

## 설치

[Chrome 웹 스토어](https://chromewebstore.google.com/detail/bjlhbdemdpjgmkpndphcfhnnhdknaple)에서 설치할 수 있습니다.

## 데이터 출처

삼성웰스토리 PlanEAT(`m.planeatchoice.net`) API를 사용합니다.
