# Privacy Policy

**Effective Date:** March 24, 2026

## 1. Information Collection
The "구미 2C 3식당 메뉴" (hereinafter referred to as "the Extension") does **NOT** collect, store, transmit, or share any personal information, user data, or browsing history.

## 2. Network Requests
The Extension makes network requests exclusively to `m.planeatchoice.net` solely to retrieve public cafeteria menu data (JSON API) and related food images. It does not send any user-identifiable information in these requests.

## 3. Third-Party Services
The Extension does not use any third-party analytics, tracking, or advertising frameworks. 

## 4. Contact
If you have any questions or concerns about this privacy policy, please open an issue in this GitHub repository: [https://github.com/acapzed/gumibab](https://github.com/acapzed/gumibab).
